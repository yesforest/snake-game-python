# 🐍 Classic Snake Game in Python

This is a classic Snake Game built from scratch using Python and the Pygame library. This project serves as a practical application of game loop mechanics, event handling, and custom grid physics during my pre-university software development journey.

## 🚀 Features
* **Custom Grid System:** Fully optimized grid math for precise collision detection.
* **Smooth Controls:** Responsive movement handling using keyboard arrow keys.
* **Dynamic Score Tracking:** Real-time score rendering on the screen.
* **Game Over Screen:** Immediate game-over state triggering upon wall or self-collision.

## 🛠️ Technologies Used
* **Language:** Python 3
* **Library:** Pygame
* **IDE:** Visual Studio Code

## 🎮 How to Play
1. Install Pygame: `pip install pygame`
2. Run the game: `python ilan_oynu.py`
3. Use **Arrow Keys** to steer the snake and eat the red food. Avoid hitting the walls or yourself!
